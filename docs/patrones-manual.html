Notas tecnicas importantes

Sin dark mode: La aplicacion solo tiene tema claro. No hay prefers-color-scheme ni toggle de tema. Fondos blancos con acentos azules.
CSS custom vs Tailwind: El proyecto usa Tailwind instalado pero la mayor parte del styling es CSS custom en globals.css (~2300 lineas). Los componentes usan clases CSS y estilos inline, no utilidades Tailwind.
Sin component library: No se usa shadcn/ui, Radix UI, ni Chakra. Todos los componentes son custom. Las clases tienen versionado (v2) indicando iteraciones de diseno.
Toast notifications: Via react-hot-toast, posicion top-right, configurado en el root layout.
